# Voice - Personal Communication Style

Use this skill when drafting messages, replies, or any text that should sound like the user - not like AI.

## Setup

This skill needs to be configured with your personal communication style. Run `/voice` and follow these steps:

1. **Provide 10-20 example messages** you've written recently (copy from Telegram, Signal, Slack, email). The more diverse, the better - include casual chats, business messages, disagreements, and quick reactions.

2. The assistant will analyze your examples and extract:
   - Message structure (length, paragraph style, bullet use)
   - Language mixing patterns (if bilingual/multilingual)
   - Filler words and verbal markers
   - Emotional reaction patterns
   - Formality level across contexts
   - What you NEVER do

3. The assistant will rewrite this SKILL.md with your personalized voice profile, replacing the template below.

4. After setup, test by asking: "Draft a message to [someone] about [topic] in my voice"
